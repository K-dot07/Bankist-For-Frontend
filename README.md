# Bankist

A modern banking landing page built using HTML, CSS, and JavaScript. The project showcases interactive UI components, smooth scrolling, lazy loading images, tabbed operations, sliders, and responsive design techniques.

## Features

- Responsive landing page design
- Smooth scrolling navigation
- Sticky navigation bar
- Tabbed operations section
- Lazy loading images for better performance
- Testimonial slider/carousel
- Modal window functionality
- Reveal sections on scroll
- Modern JavaScript DOM manipulation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/bankist.git
```

2. Open the project folder:

```bash
cd bankist
```

3. Run the application by opening `index.html` in your browser.
